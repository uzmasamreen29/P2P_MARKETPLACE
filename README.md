# Peer-to-Peer Tutoring Marketplace

## 📌 Description
A web-based tutoring platform where students can search tutors, request sessions, and rate them after completion.

## 🚀 Features
- Student & Tutor login
- Slot management
- Session booking system
- Request/accept flow
- Session completion tracking
- ⭐ Rating system (average + per user rating)

## 🛠 Tech Stack
- Flask (Python)
- MongoDB
- HTML, CSS, Bootstrap
- JavaScript

## 📊 Project Flow
Request → Accept → Book → Complete → Rate
